# Client side code for AI Service Assistant App

This part assumes that you've gone though main `README.md` file of the repository.

## Getting started

Most important commands are listed in `Makefile`. To learn about their usage just run:

```
make help
```

## Simulator environment

Download and install [Android Studio](https://developer.android.com/studio) and find some guides how to setup virtual device. When you have your simulator running, use:

```
make android
```
