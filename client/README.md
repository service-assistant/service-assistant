# Client side code for AI Service Assistant App

This part assumes that you've gone though main `README.md` file of the repository.

## Getting started

Install required dependencies:

```
npm install
```

## Linting / formatting

To format code you can use:

```
npm run format
```

To verify all code formatting and linting is fine run (single command):

```
npm run lint
```

## Testing

All tests should be put inside `__tests__` directory.

To run all tests you can use:

```
npm run test
```
